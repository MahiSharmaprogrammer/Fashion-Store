# 👗 Fashion Store Website

A responsive Fashion Store website built using **HTML** and **CSS**. This project is designed as a simple e-commerce homepage that showcases fashion products, shopping offers, festival discounts, and payment options.

## 🚀 Features

- 🛍️ Attractive homepage design
- 👕 Product categories (T-Shirts, Shirts, Shoes, Pants, Sunglasses)
- 🖼️ Shopping banner section
- 🎉 Upcoming festival sale table
- 💳 Payment options section
- 📱 Responsive layout
- 🎨 Clean and simple user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
